Introduction

LumenCanvas is an open source HTML5 drawing tool based on Fabric.js & JQuery. You can use it to embed drawing boards in web pages. You can save the drawings as a string or JSON and you can retrive it again. LumenCanvas is an object based drawing tool. You can move the drawn objects around and change the border of the filling color.



Lumen Canvas’s features:

  • Basic drawing tools
  • Panning and zooming
  • Background images/shapes and watermarks
  • Retina support
  • Saving and loading JSON
  • Exporting drawings to PNG and SVG
  • Events

Installation

<html>
    <head>
        <!-- dependency : JQuery -->
        <script src="js/jquery/jquery-1.12.4.js"></script>
        <script src="js/jquery/jquery-ui.js"></script>
        <link rel="stylesheet" href="//code.jquery.com/ui/1.12.1/themes/base/jquery-ui.css">
        <!-- dependency : FabricJS & Spectrum plugin -->
        <script src="js/fabricjs/fabric.min.js"></script>
        <link rel="stylesheet" href="css/spectrum.css">
        <script src="js/spectrum.js"></script>
        <!-- LumenCanvas -->
        <link rel="stylesheet" href="css/lumen-canvas.css">
        <script src="js/lumen-canvas.js"></script>
    </head>
    <body>
        <div id="drawing-canvas" class="drawing-canvas toolbar-at-top" style="height: 600px; margin: 0 auto;"></div>
        <script>
            new LumenCanvas ({
                selector : "#drawing-canvas"
            });
        <script>
    </body>
</html>

JSON Output

/// Create new instance
var paintInstance = new LumenCanvas ({
    selector : "#drawing-canvas2",
    showGridButton : false,
    enableGridByDefault : false,
    watermarkImage : "img/watermark.png",
    watermarkImageOpacity : 0.8,
    width : $("#drawing-canvas2").width() - 65,
    defaultActiveTool : "Line",
});
/// Export SVG
$("#export-svg").on("click",function(){
    $("#svg-output").val(paintInstance.GetSVG());
})
/// Export JSON
$("#export-json").on("click",function(){
    $("#json-output").val(JSON.stringify(paintInstance.GetJSON()));
})
